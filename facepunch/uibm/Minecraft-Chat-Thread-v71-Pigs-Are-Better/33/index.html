anyone got any tips on castle building? my table looks like shit and my rooms look too empty
[QUOTE=Zukriuchen;40872230]anyone got any tips on castle building? my table looks like shit and my rooms look too empty[/QUOTE] The first step in any great castle build is to build a castle. I hope that helps.
For tables you have a variety of options, you can use stair blocks and slabs, upside down pistons, among others. If the table is very large you could try a pattern using birch and spruce. You can make empty rooms look more filled by making recesses in the walls, rather than one long flat wall. Make a house banner along one wall using wool blocks, or make it on the floor instead. A big roaring fireplace in the main hall. Make frames around windows a different material to the rest of the castle to make them stand out. Experiment with depth, it makes things look 10x better than flat square shapes.
