The current state of Minecraft according to a friend that's worked with Grumm by coding some examples is basically that they've made it so that mods will be much harder to make, as you can't render stuff on top of blocks anymore. Every single instance and possible appearance of a block must have its own model file, which even for a mod like Buildcraft with their pipes, with the positions, colors, wires and gates, each and every possible configuration would need its own model, would not only be a pain in the ass but also incredibly inefficient (but according to him, redstone dust is the exception. Not even MOJANG wants to fuck with that system. What does that tell you?). He's also not heard a word from Grumm since the buyout, and there's talk that Microsoft wants to switch Minecraft to .NET. (not sure how true the .NET part is)
