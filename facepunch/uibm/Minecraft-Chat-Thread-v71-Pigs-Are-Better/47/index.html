I could google it but I figure I'm doing something very simple very wrong and thought I'd ask here first since googling minecraft related stuff usually yields 100000 results in the wrong places.[/QUOTE] Install the latest Minecraft Forge (click on installer): [url]http://www.minecraftforge.net/forum/index.php?action=files[/url] Run the jar file once you've downloaded it. It'll set up a separate folder for itself in your versions folder. Run minecraft, edit your profile and select forge as the version you want to use. create a folder inside .minecraft called "mods", put the .jar files/.zip files of the mods you want to use inside there. And you're done!
