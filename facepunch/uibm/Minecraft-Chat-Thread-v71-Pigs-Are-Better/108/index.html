[b]mc.nodex.net[/b] [u][b]Server switching/selecting[/b][/u] When you first join Nodex, you’ll be in our hub (/server hub), from there, you can walk through any of the portals to switch servers without having to go back to the main menu, or you can type /server to see all of the servers available, then type /server [server name] to go to that server. It’s somewhat common sense, /server creative will go to the creative server, /server quake, /server factions and so on. [b][u]Friends[/b][/u] We’ve implemented a custom friends system onto all of our servers, each server is linked together using this system. To add someone to your friends list, just type /friend [player name], to accept a friend request, do the exact same command, only with the person who sent you the request’s name as the player name. Once you’re friends with someone, you can teleport to each other as long as it’s on a server which teleporting won’t make the game unfair (E.G, factions won’t allow TP), /ftp [player name] [b][u]Messages[/b][/u] We’ve got the classic essentials /msg, /m, /r and such, but on top of that, we’ve got our own custom messaging system. Once you’re friends with someone you can message them, no matter what sever they’re on, just do /pm [player name] [message] (We’re adding /pmr soon, this will act like /r in essentials does). You can do /fm [message] to send a global message out to all of those on your friends list, yet again, they can be on any server. We’ve got our own forums, [url]http://forums.nodex.net/[/url] and a Minecraft website (made by me, yay!) [url]http://mc.nodex.net/[/url].
