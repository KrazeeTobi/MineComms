Incoming mass of images and explanations!: [highlight]Before anyone asks[/highlight], the texture pack I am using is [url=http://www.minecraftforum.net/viewtopic.php?f=25&t=25676]dokus' RPG texture pack[/url], Teacarp is going to be posting some screenshots (probably less than what I have) with the vanilla texture pack later on. [b]Credits:[/b] Main Contributors: Teacarp Blackout phabeZ With the occasional help from: King Kambrook Crivens McJugs! (and sorry if I missed anyone) [b]The Rapture[/b]: We never established a true name for this, except overall it was greatly inspired by Bioshock's underwater city, the Rapture. So as a result, we have always referred to it as the Rapture. In creating this, it was very difficult for obvious reasons (being underwater). We tried to make it very private and also difficult to get into, whilst maintaining an aesthetically pleasing appearance. I think the execution of this was done well, there is only 2 ways (excluding destroying the walls of stacking blocks) which are both very hidden, and one involves a huge tunnel which the entrance a long way away. There are several secrets within this too. For the sake of not stretching the page too much, I will thumb these. They're in jpg format and 1920 x 1080 resolution, so you can decide whether you want to view them in their full size. This is the entrance. To get in, you have to find a 1x1 opening, swim down and the forward. It is very hard to locate from the surface, and while not perfect in how it was made it certainly is sufficient in keeping unwanted visitors out.
