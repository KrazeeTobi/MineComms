First start scaffolding like usual, use dirt blocks for the first 2 blocks. Than switch to sand. Once you have used that scaffold and need to destroy it, get down and break the first dirt block so you have a dirt block floating in the air. Place a torch there. Break the other dirt block. All the sand/gravel falls, but because of the torch it breaks, thus destroying your scaffold with ease.
