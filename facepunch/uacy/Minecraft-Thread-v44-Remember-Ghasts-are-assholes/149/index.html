[highlight][B]Instructions for running a Minecraft server with increased memory:[/B][/highlight] First thing you need to do is download and install the proper version of the Java SDK for your OS. The installer can be found [URL="https://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_Developer-Site/en_US/-/USD/ViewProductDetail-Start?ProductRef=jdk-6u22-oth-JPR@CDS-CDS_Developer"]here[/URL] After that you need to make a batch file with the following commands: [code]java -Xms1024M -Xmx4096M -jar minecraft_server.jar[/code]1024 is the minimum amount of RAM to give your server to use (in MegaBytes) and 4096 is the max. So a server using this exact batch file would start out at 1GB of RAM dedicated to the server and would scale up to 4GB if needed. Note: In case you don't know how to make a batch file, just make a text file with the needed commands. Save it in the same folder as your Minecraft_server.jar (or whichever server mod you are using, in which case you need to replace minecraft_server.jar in the command with the name of your mod.). Then replace the .txt extension of the file with a .bat extension.
