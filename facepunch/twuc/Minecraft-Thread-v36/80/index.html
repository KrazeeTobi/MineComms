[code] #NoEnv #SingleInstance force #MaxThreadsPerHotkey 1 #InstallKeybdHook #InstallMouseHook SendMode Input CAPSLOCK::Suspend , Toggle $MButton:: send, {Lbutton Down} Return $q:: loop { if getkeystate("q", "p") ;if a button is Physically held down by the user. { send, {q DOWN} sleep, 50 send, {q UP} } else { break } } return [/code] It also makes Q rapidly drop stuff, but you can take out everything after the first return if you don't want it.
