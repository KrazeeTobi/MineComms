[QUOTE=Benie;18460707]Is there a tutorial anywhere to use Omen? I'd like to learn how to make my own maps.[/QUOTE] It's really not that complicated. If you want a new map, click File>New, and there you can select the X-size, the Y-size, and the Z-size. Then, you can choose to generate a map using certain presets, like mountains, or desert, etc. You can also choose to create a flatmap with many surface materials. Then, you scroll to go to different layers. The buttons at the bottom that say 'X-Y layers' or something like that let you choose which side you are viewing it from. When you're on a layer you want to edit, you can choose either a square brush or a round one, or you can use tools like a line tool or the flood fill tool, (3D replaced the clicked material with the selected one in 3D). Not sure what else is so complicated. After all that is done, you can save the map, and load it with your server software.
