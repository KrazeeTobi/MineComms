[QUOTE=KStyleAzure;23974184]How do I edit my inventory using the persister?[/QUOTE] From mc forums First you will need to download this: [url]http://kingsandwich.com/mc/persister.jar[/url] which lets you keep your inventory on a server when you disconnect, you will still lose it if you die. Next just run persister and join a server of your choice, then after a few seconds disconnect. Now press windows key & r on your keyboard and run %appdata% find the .minecraft folder which is in the "roaming" folder. Once there you will find some files called something like mpsave_*server ip*. Open the one which matches the ip of the server that you were on in notepad. Delete what is in the file and paste this over it for a ready to go inventory: [code]4,64,0;10,64,0;259,64,0;277,64,0;278,64,0;279,64,0;46,64,0;46,64,0;46,64,0;46,64,0;8,64,0;7,64,0;57,64,0;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;!;[/code] Now save that file and exit notepad. Right click on the file that you just edited, click properties and tick the box that says "Read-only" now click ok. You can now join that same server again and you should have some nice items :) (you may need to wait just a few seconds for it to appear) It may not work if you die before the world loads, but if you do die and lose your inventory just rejoin the server.
