If anyone with a server wants to create a Cartograph image every x seconds for a timelapse or something I cooked up this batch file Put it in a .bat file in your server folder Set the correct path for Cartograph and maybe customize the render settings [code] ::Makes a cartograph map of the server map every x milliseconds:: @echo off SET Start=%CD% IF NOT EXIST Carts ( mkdir Carts ) ::Delay in msec:: SET Delay= 30000 ::Path of Cartograph directory:: pushd C:\Cartograp @GOTO Run :Run set hh=%time:~0,2% if "%time:~0,1%"==" " set hh=0%hh:~1,1% set yy_mm_dd__hh_mm_ss=%date:~12,2%_%date:~4,2%_%date:~7,2%_%hh%_%time:~3,2%_%time:~6,2% echo Creating map %yy_mm_dd__hh_mm_ss%.png start /low /b Cartograph 6 normal 0 0 0 0 %START%\Carts\%yy_mm_dd__hh_mm_ss%.png %Start%\world ping 1.1.1.1 -n 1 -w %Delay% >NUL @GOTO Run ENDLOCAL GOTO:EOF [/code]
