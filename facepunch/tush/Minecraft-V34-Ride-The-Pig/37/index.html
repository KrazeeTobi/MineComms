[highlight][b]Setting up a server for SMP, dawgs[/highlight][/b] First, you'll need the server software, [url=http://www.minecraft.net/download.jsp][u]you can get that here.[/u][/url] Then, you'll need to download [url=http://www.simpleportforwarding.com/download][u]Simple Port Forwarding[/u][/url] This tool makes port forwarding very easy. Now, we need to make sure you have a static IP, [url=http://portforward.com/networking/static-xp.htm][u]follow this handy guide to do so[/u][/url] (this should work across all Windows versions). When I make my IP static, I always set it to from 10 to 30. Now, on to the port forwarding program. Your program should look a little something like this (they changed it recently so it's a eyesore, but everything is in the same place) [img]http://imageflock.com/img/1246912161.jpg[/img] Select your router where it says "((Select your router))" You can find your router type by entering [url=192.168.1.1][u]192.168.1.1[/u][/url] in your browser, and the default username and password should be something like admin and admin. So, select your router that the router page says it's model name For me, it's on the far left. [img]http://ahb.me/fbu[/img] Now, select your router. [img]http://imageflock.com/img/1246912304.jpg[/img] Now, goto File > Add new port You should see something like this window [img]http://imageflock.com/img/1246912366.jpg[/img] Click the custom tab, and enter [img]http://ahb.me/fbw[/img] Except, in the static IP box, enter what you set your static IP ending to be. and now click add, and your port shall be forwarded! Now, to check if it's actually working Go to Tools > Test your port forwarding [img]http://ahb.me/fbz[/img] Enter what is in the "Port #" box and then click add, and then begin. and if you see this [img]http://ahb.me/fbA[/img] Then you're good! Now, open up the server software you have (for beginners, use the Minecraft_Server.exe vanilla version). Let it generate the map, and say it's finished in the box. [img]http://ahb.me/fbC[/img] Then you're done! Go to [url]http://www.whatismyip.org/[/url] and copy and paste what it gives you into the multiplayer text box. If all went well, you connect! Now share it with your friends! (By the way, no need to add the port to the end of the IP) PS. Thanks to [url=http://www.minecraftforum.net/memberlist.php?mode=viewprofile&u=25654&sid=95258926d80f656563920c57d99303f2][u]Skype[/u][/url] at the MC forums for a lot of the images!
