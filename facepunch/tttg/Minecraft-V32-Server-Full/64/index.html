There are thousands more cases where this is annoying but you get the picture. I think this could pretty easily be solved by making right click ONLY place / use the block you are holding and interacting with the environment (doors, carts etc) down to a separate button. The use key would affect the following items: > Furnace GUI > Chests > Doors > Carts/Boats > Switches Pretty much anything that you interact with in the environment. I imagine a lot of people are going to disagree with the importance of this but as minecraft gets more and more complicated with more blocks you interact with I think people are going to regret not separating the functions of right click pretty quickly.
