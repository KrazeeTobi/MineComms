I use texture packs the entire time, and receive no errors. I also use [b]no signs, EVER[/b]. The problem has been reported countless times on the official minecraft forums, and all signs (hehe, double-entendre) point to the signs being a problem. If you have any signs in your map after crashing, try using NBTForge and removing them if possible. And NBTEdit to remove them from inventory if you are still crashing. If this doesn't work, then I don't know what will. Beware: Images ahoy. Here's an example way to install sami-pso's custom texture pack, using the Brocraft/MultiCraft client... First, Download the pack. Sami-pso's latest was found at: [url]http://www.mediafire.com/?ygzcnizmlji[/url] If you're using WinRAR, it will look like this: [img]http://img266.imageshack.us/img266/1841/minecraftaddskins1.png[/img] Just extract the sami-pso folder into the texturePacks folder, the DS store files are for Mac's and can be deleted. [img]http://img824.imageshack.us/img824/4594/minecraftaddskins2.png[/img] Then go through each folder deleting the unnecessary extra files. Such as the 'thumbs.db' and '.DS_Store' files. [img]http://img194.imageshack.us/img194/3661/minecraftaddskins3.png[/img] Now create a new zip file, and name it whatever you want. It helps if you keep it relative to the texture pack. Then add the files to that zip. Be sure not to add extra files such as the "Alt Glass" folders, because they are extras and will not be used. [img]http://img685.imageshack.us/img685/273/minecraftaddskins4.png[/img] Another thing you need to watch out for, is to make sure you do [b]NOT[/b] extract the entire folder, like so. You cannot have them inside a folder inside the zip. [img]http://img88.imageshack.us/img88/6397/minecraftaddskins5.png[/img] Then when you run MultiCraft.exe, you will be prompted with the login screen. At the bottom of that, you change the texture type like so: [img]http://img171.imageshack.us/img171/2562/minecraftaddskins6.png[/img] Happy minecrafting! [img]http://img163.imageshack.us/img163/4421/minecraftaddskins7.png[/img]
