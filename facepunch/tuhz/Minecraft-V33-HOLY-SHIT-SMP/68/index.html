[*]/item command - gives you items (can be toggled between admin-only and everyone, you can also supply a whitelist and blacklist) [*]/playerlist command - shows you the current playerlist [*]/kick and /ban commands (remember that /ban bans IPs!) [*]Since you can spoof names, I added IP checking for admins. If you don't have the IP corresponding to the name specified in admins.txt you don't get access to admin commands. [*]Teleport command.
