[QUOTE=DireAvenger;25951163]Someone modded the sand to fly upwards instead of falling downwards. He didn't release it, but he posted screenshots here and it looked hilarious.[/QUOTE] Hello. The sand is not actually falling, though. It works a bit strange. When a sand (or gravel) block detects that there are no blocks under it, it destroys itself, and creates a new entity, EntityFallingSand. THis is basically a walk-through block (more like a block-shaped particle) with the texture of the block that created it, and creates that same block when it detects it has 'hit' the ground. In other (more to the point) words, I can make the dirt and grass fall down, and I'll do so right now. It may take a bit longer, since I have never released anything yet (testing the code works different than distributing it). But the good news is that it will not affect as many blocks as the sand ever did, so it should run on normal computers.
