[QUOTE=VistaPOWA;25854308]Could someone tell me how to enable AA in Minecraft using an NVIDIA card?[/QUOTE] On Vista or Win7 (not sure about XP) right click on the desktop and choose nVidia Control Panel. From there you need to choose Manage 3d Settings. Then click the program settings tab and click add. You need to find and add javaw.exe. You'll need to set antialiasing - mode to override any application setting, then you need to choose your antialiasing - setting options and if you have a card new enough to support it, set antialiasing - transparency as high as you can. (The lines disappear for me at 8x supersampling.)
