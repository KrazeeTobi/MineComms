For fucks sake guys, you don't need Hamachi for LAN play, honestly, everyone thinks this, they are wrong, you don't need a stupid external program. It's better performance than hamachi if you run off router anyways, so... Okay, lets move on.. make a server with the regular bukkit client, or the regular server if you're not wanting to use bukkit. Then, if you want to connect yourself, type localhost in ip box for you, but for your brother, run command prompt, and type ipconfig, it should say "Default Gateway: 192.168.*.*/10.0.0.*" and use that as the ip for your brother if you're hosting, and vice versa. However, since I am not sure if that works, you can go on your computer and do ipconfig, and look at the "IPv4 Address: 192.168.*.*/10.0.0.*", use that as the ip for your brother. The IPv4 thing always works, so try that if the default gateway IP doesn't work.
