Thanks everyone else for the help though.[/QUOTE] Me and my sister play minecraft over LAN, so I think I'm qualified to help. First, install minecraft on both PCs. Then, download the minecraft server executable. Put it in it's own folder, then run it once so it generates the approriate files, then exit. In the 'server.properties' file that has been created, change the variable 'online-mode' to false. Run the server again, this time keep it open. Open minecraft (on the same PC as the server), sign in with your account, then click multiplayer and type 'localhost'. You should connect to the server. Sign in using invalid details on the minecraft on your brother's PC, then click the 'Play Offline' button. Click 'multiplayer' and enter the name of the computer the server is running on (probably something like 'Hayden-PC', look in the 'system section of your control panel if you're not sure). Click connect, and you should have joined the game. This seems like a lot of work but it really is quite straightforward.
