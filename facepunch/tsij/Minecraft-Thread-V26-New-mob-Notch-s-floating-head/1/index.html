It should be in a .rar file. #2: Open the texture pack in WinRAR. #3: Go to %appdata%\.minecraft\bin\ and right click minecraft.jar --> open with --> WinRAR #4: The files in the texture pack should be in a structure with terrain.png in the root folder (it should be visible when you open the .rar) and folders such as armor, mobs etc. #5: If #4 is true, Drag/Drop the texture files from the pack to minecraft.jar (open in winRAR) #6: Run minecraft! NOTE: Your textures will be wiped if Notch updates terrain.png, so be prepared to re-install packs often. [B]I have installed a texture pack - some objects are pink/do not appear at all[/B] Your texture pack is not up to date. Wait for the creator to post the latest version. Alternatively, create the new textures yourself (edit terrain.png and the other files) [B]How do I create my own texture pack?[/B] (Thanks to DeanWinchester for this answer!) 1. Get WinRAR 2. Go to "%appdata%\.minecraft\bin\minecraft.jar" 3. Open "minecraft.jar" with WinRAR 4. Create a folder somewhere called "Minecraft Textures" 5. Extract all png/gif files in there, as well as the folders: "armor" "art" "gui" "item" "misc" "mob" "terrain" to the folder you created. 6. Edit them and save. 7. Drag and drop everything from your "Minecraft Textures" folder back into the jar file that is opened with WinRAR (following the same folder structure) and let them replace the others. 8.
