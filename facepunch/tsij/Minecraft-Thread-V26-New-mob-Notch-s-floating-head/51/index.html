Here's the list I kept while learning: [code]Shortcuts List: Tab: Toggle edit mode ** Ctrl+Tab: Change edit selection mode (vertices, edges, faces) ** Shift+Space: Fullscreen panel Ctrl+Up: Alternate Fullscreen panel Num5: Toggle ortho/perspective Num1,3,7: Ortho views (front, top, etc.) Num2,4,6,8: Rotate view Ctrl+Num2,4,6,8:Pan view Shift+C: Center target on object, then: C: Center view on target A: Select All/Deselect B: Box Select B+B: Circle Select ** Ctrl+Hold LMB: Draw lasso selection D: Change draw mode (wireframe, opengl solid, etc.) Z: Toggle wireframe mode (can't use Limit Selection to Visible with wireframe on) ** F: Create edge/face (needs 2-4 vertices selected) Ctrl+LMB: Create Vertices (in edit mode) W: Specials menu - subdivide, remove doubles, merge, Alt+M: Merge vertices R: Rotate S: Scale G: Translate Z/X/Y: Constrain axis during transform ** X/Del: Remove Selected Face/Edge/Vertices ** Shift+S: Snap to grid U: UV Unwrap / Other Settings E: Extrude MMB: Alternate axis of extrusion while extruding Ctrl+Space: Adjust 3D Transform Manipulator F12: Start Render[/code]
