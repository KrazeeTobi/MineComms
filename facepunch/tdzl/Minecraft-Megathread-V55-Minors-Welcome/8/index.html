[IMG]http://i.imgur.com/LhV3Q.png[/IMG] [b][url=http://www.mediafire.com/?0k77j8zfvd5a0xa]Download the latest version on the Medieval Craft [32x32] Texture Pack, 1.0.1b![/url][/b] Note: The terrain.png is completely useable, the b is to indicate we're not done with the pack yet! Read below under "To Do" to see why. [b]About[/b] A texture pack loosely based on a Medieval style setting. Initially was called the "Ancient Greek pack" or "Hellenic pack" however after continuously straying away from that theme in order to keep it interesting, we decided it was no longer an 'Ancient Greek' pack. This pack was designed for Multiplayer use, however it will work fine in Singleplayer. [b]Installation[/b] [i]If you want to use any of the Alternative textures, read under the "Alternative Textures" sub heading![/i] 1. Download the .zip file above 2. If you don't want animated water + lava and you don't mind some blocks not displaying correctly, skip this step. If you want to use the texture pack as best as possible, download [url=http://www.minecraftforum.net/viewtopic.php?f=1021&t=46173]xau's HD Texture Fix[/url], if you have trouble using it request help in that thrad, not this one. 3. Paste the .zip file (do not extract it!) in your Minecraft texturepacks directory, you can either navigate to it manually or access it via the ingame menu. 4. Select the texture pack in the "Mods and Texture Packs" window 5. Enjoy! [b]Alternative Textures[/b] If you wish to use a texture other than the one in the original terrain.png, follow the following instructions. The block which is probably furthest from the original is the lightstone, which has been changed to be more practical in building situations. There are however 2 alternative versions that have a more natural look. 1. Extract Medieval Craft.zip 2. Open both the Alternatives.png and Terrain.png in your image editor of choice 3. On the Alternatives.png, highlight via a select tool the texture you would like to replace (it has to be the exact 32x32 pixels!) and copy it (shortcut Crtl + c) 4. Open the window with the terrain.png file, and paste the your recently copied texure/s down and place it over the texture you're wanting to replace. 5. For textures which have a transparent background, it will probably require erasing the original image behind it. Either use the eraser tool, or select the area and press 'Del'/'Delete' on your keyboard. If you are unable to do the above, post in the thread nicely asking and specifying your requests, and someone may be nice enough to do it for you. Please don't ask just to be lazy!
