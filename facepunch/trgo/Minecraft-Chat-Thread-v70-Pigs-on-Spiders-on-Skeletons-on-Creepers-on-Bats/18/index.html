[QUOTE]This mod is still in it's early stages, and due to various issues I haven't been able to work on it as much as I'd like, but I wanted to get the basic idea out there and get a feel for the communities opinion. At it's core, this mod will allow for a more immersive and mechanized crafting experience, but still requiring the player to work for it. It will essentially allow for most items and blocks to be melted down into a more liquid form, which you can then store using whatever liquid storage system you choose. But what's storage without a purpose? The main purpose of melting these things down is to eventually pour them into specialized machines which will allow you to create items out of their liquid components, saving you some resources, and giving you a potential automated crafting environment. Unfortunately, the changes in 1.5 have caused me to have to reconsider how some things will end up working, but work is steadily being done. What I want from you all is ideas and feedback on what we already have. You can find our current progress [URL="github.com/dmillerw/Foundry"]here[/URL], in our GitHub repo. I'll eventually provide instructions on how you can test the mod for yourself, and even develop with us.[/QUOTE] Thoughts?
