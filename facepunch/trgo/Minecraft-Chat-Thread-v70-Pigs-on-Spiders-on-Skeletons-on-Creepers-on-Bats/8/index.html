[QUOTE=Oicani Gonzales;39843530]i made a worldpainter world, and everything outside it is ocean, forever. is there a way to make it not be ocean forever? changing the seed in the level.dat didnt work[/QUOTE] [URL]http://www.worldpainter.net/trac/wiki/FAQ#HowdoImakethebordersofthemaplookniceandintegratewellwiththeMinecraftlandscape[/URL] Hope this helps. I also did a quick test. When you come to export map I chose a random seed under the general tab and ensured that it was set no 'no border' However, as you probably expect doing this will generate the random seed land around your square map. So smoothing it out with WorldEdit will take some time. I wouldn't say this was the best option of using worldedit to fix it, but in the FAQ it suggests another lengthy but more visually appealing process of fixing it [QUOTE][IMG]http://i.imgur.com/bIWd8I2.jpg[/IMG][/QUOTE]
World Painter doesn't have 'normal' generation outside of the map borders. It only has the pre-defined settings of void, water, lava, or bedrock. But from my experience of using worldpainter if you still have the original worldpainter file then you can edit the border properties by hitting CTRL+P. From here this will be the best fix. like I mentioned you can change the border from 'water' to 'no border' and set a random seed generation in the bottom box. From there you could either merge in the seed border with your map border with world edit in game. He also suggests in the wiki FAQ to load the borders of your map in minecraft by flying around. From there you import it back into worldpainter. The map will contain part of the loaded seed border and all you have to do is use the smooth tool to merge the terrain. Export it back into minecraft, see what it looks like, and then the world will continue to load the seed as you fly around. I hope this helps, and kind-of makes sense to fix your problem.
