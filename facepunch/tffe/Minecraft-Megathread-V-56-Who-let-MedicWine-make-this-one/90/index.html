[QUOTE=Oicani Gonzales;27853369]How do I make a plane? I never had shadows :saddowns:[/QUOTE] If you're using 3DS Max: 1. On the "Create > Geometry" tab, create a plane. Change it's colour to a slightly off-white. 2. Add an "edit poly" modifier and extrude the side and back edges upwards to form a sort of "stage". 3.Select all the faces and use "Tessellate" in the Edit Poly modifier to cut it up a bit. 4. Add a "Turbosmooth" modifier to curve it nicely around the bottoms. 5. Go to "Rendering > Light Tracer" and make sure the dropdown is on "Light Tracer". 6. Change "Bounces" to 1. Also make sure your light has it's Shadows "On" box ticked (and that you have a light in the first place). If you're using blender: Feck knows, I'm too lazy to try and understand it's weird-ass controls and interface.
