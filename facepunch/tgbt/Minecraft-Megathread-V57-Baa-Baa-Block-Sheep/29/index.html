[b]Hey GUYS[/b] THREEEEEEE DEEEEEEEEEEEEEEEEEEEEEEEEEEEEEEEEEEEEE [media]http://filesmelt.com/dl/2011-02-10_20.25_.16_.png[/media] And here is a guide on how you can make YOUR minecraft look 3D too! Step uno: Download the GLSL shader mod hereski [url]http://www.minecraftforum.net/viewtopic.php?t=128043[/url] and install the parallax shader mod (remember to remove DoF as it breaks it on ATI cards) Step due: Install Aza's arid pack (Currently the only pack with a decent terrain_nh) and patch it. Step tre: Find a set of lines that looks like this in the parallax mod [code] // These numbers might need to be tweaked per texture pack. In particular, the third number is inversely proportional to the range of the height-map. vec3 interval = viewVector * vec3(0.0039, 0.0039, 4.5); // Fine for 16x16 tile size //vec3 interval = viewVector * vec3(0.0019, 0.0019, 0.5); // Fine for 32x32 tile size //vec3 interval = viewVector * vec3(0.00048828125, 0.00048828125, 0.2); // Fine for 128x128 tile size[/code] and comment out the first vector interval, whilst uncommenting the third vector interval. Step quattro: Almost there! Put this [url]http://d.pr/aiMw[/url] terrain_nh.png (remember to name it appropriately) into your minecraft.jar and run the game. Tada! You should have parallax! For ATI users, there are a couple of extra steps. First install catalyst control center and it's appropriate drivers, the latest version being 11.1 which is required for GLSL compatability. DISABLE CATALYST AI. IMPORTANT. And remember to check OpenGL Triple Buffering. Be on the wait, the GLSL mod's soon to get dynamic lighting and improved specular mapping light sources.
