Alright, to everyone that wants to start minecraft on steam, but it doesn't start right or whatever, I'm going to help you out. First, make sure javaw is in your PATH variable, you can do that by opening cmd and typing "javaw". It should look like this if it's in your path: [Img]http://i.imgur.com/Hfh8C.png[/Img] If not, then you will have to do a couple more steps. If it is, then go ahead and continue to the setting up with steam part. [b]Setting up the PATH variable[/b] First, search for environment in the start menu: [Img]http://puu.sh/5XNL[/Img] Choose user account, a window should pop up. Just click "Environment Variables". You should get something like this: [Img]http://puu.sh/5XOc[/Img] You can choose either one, the only difference is if you choose account instead of system, then javaw will only work for the current user account. Choose either user or system depending on where you want the javaw variable to work on, then scroll to path and press edit. [Img]http://puu.sh/5XOy[/Img] As you can see there, I already added javaw for java7 to my path. I'm running 64bit java so my java is in just Program Files (I'm also on 64bit windows), it's usually in Program Files (x86)/java/<version>/bin, or if you're on 32bit, Program Files/java/<version>/bin . All you have to do is find the exact path to the bin folder for your version of java, and enter it after the end of the string. Make sure to add a ; to the end of the current string. So say the current string was: 'abc', and you wanted to add: 'xyz', it would look like this: 'abc;xyz'. Be very careful not to remove anything of the existing string, as some programs need the path to function. After adding javaw to the path variable, check to make sure it worked. If it did, then continue to the next part. [b]Setting up minecraft for steam[/b] When javaw is added, what you will want to do is add minecraft.exe to non steam games, if you haven't already. Then right click the entry in steam and select properties. Then you will want to set the target to this: [Code]"javaw" -jar "<PATH TO MINECRAFT.EXE>"[/Code] If you want the launcher to auto log you in, then just add your username and password to the end, after the quotes. So it will look like: [code]"javaw" -jar "<PATH TO MINECRAFT.EXE>" USERNAME PASSWORD[/code] It should launch and say you're in game until you close minecraft.
