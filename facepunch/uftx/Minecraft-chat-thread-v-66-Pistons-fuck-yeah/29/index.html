[QUOTE=Zoran;31262126]The cool thing is I don't fucking know how to allocate it.[/QUOTE] -Xmx and -Xms switches in java. You can use the Minecraft.jar directly, or use the Minecraft.exe through a batch file or a shortcut. "C:\Program Files (x86)\Java\jre6\bin\javaw.exe" -Xmx100M -Xms100M -jar "C:\Program Files (x86)\Minecraft\minecraft.exe" I personally use the shortcut, however for some reason auto-login only works if you allocate 1000MB or greater, which will cause out of memory errors again.
