I'm making a mod that allows you to "print" items into the game (even in survival multiplayer!) How it works is you load up a "model" file (currently only handmade) that is just a collection of blocks. Then you type "print" and it then starts printing from your current location (it sets this as the bottom left.) How it prints is it just makes the player build in a 4x4 radius around the player. The structure size can theoretically be infinite because it takes note of where you STARTED the printing, so when you move around it prints the right blocks. Kind of hard to explain but it's really cool.

It currently works 100% with .schematic files, the only thing that does NOT work is metadata(so cloth, etc. will not function properly). I'm looking into WHY currently but I'll let you know. Video: [media]http://www.youtube.com/watch?v=sf3gsIByYn8[/media] (It's still processing, give it a bit). This mod allows users to "print" out schematic files into SMP servers. This does not require admin, any user that has the ability to place blocks can print out schematics. The printing is done in a 4x4 grid around the player, so to print out larger objects more movement is required. Schematic files are created in MCedit and likewise. NOTE: In order for a block to be placed, there MUST be a block ADJACENT to where it will be placed!
