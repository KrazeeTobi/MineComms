Alright, you guys remember that game I posted a few pages ago? Well, I remade it because I wasn't entirely happy with how it turned out. First because I had to manually reset it after each round, so I'd have to break the doors down, un-flip the switches, re-fill the dispensers... and that all took too long. Second, the arena was way too tiny for battle. And third, the dispensers only gave one sword for 2 people on a team. So after I re-made it, all these were fixed/changed and now I'm at least 90% satisfied with the end result. Here's the changes and some new pictures: Remember that tiny "lounge"? Well, since I made the arena itself bigger I had to expand the lounge too because the team rooms connect directly to their platforms below. It felt too empty so I added some fake couches, a table with some cakes, and a wall in the middle so I could put the game rules and some paintings on the other side. And finally, here's the end result: [IMG]http://i.cubeupload.com/mNwhVY.png[/IMG] [IMG]http://i.cubeupload.com/En8UfK.png[/IMG] The team rooms still remain the same, except their wiring has been completely redone. First off, the dispenser now gives out two swords instead of one so each team can have 2 people. Second, instead of a lever, the "ready" mechanism is now a stone button wired to a piston-based RS-NOR latch for easy reset. Here's a picture: [IMG]http://i.cubeupload.com/l5WsV5.png[/IMG] As for the observatory, I also made it bigger so it was the same width as the arena + moved the glass one block out, and this is how it looks: [IMG]http://i.cubeupload.com/6D3fF9.png[/IMG] [IMG]http://i.cubeupload.com/rfnJhV.png[/IMG] Also, that tiny arena with 9 platforms? Now it has 20 platforms to accommodate all 8 players. [IMG]http://i.cubeupload.com/2yJWFF.png[/IMG] And now for some behind-the-scenes, this is the admin entrance: [IMG]http://i.cubeupload.com/35L0xV.png[/IMG] Once you go in, you can either choose to enter the arena via a maintenance access (not pictured) or go push the "end round" button. The way it works, basically, is that since the arena is bigger, I also had more space for the wiring. So instead of doing levers I did buttons with piston-based RS-NOR latches, which could easily be reset. When you push the button, the floor on the team rooms closes up, the doors open up again and a dispenser drops 2 diamonds (1 for each team member) in the middle of the arena. I'm thinking about hosting a server on the 7th for me and a few friends to play, but there's still not enough people to fill all the 4 teams so I'd gladly make a public server so we can play for a bit.
