[QUOTE=Ziirxia;33921981]Why does it make any difference if there's grass on or not?[/QUOTE] The BUD (Block Update Detector) I'm using relies on a bug with pistons which allows them to be powered by a block diagonally above them. However this does not extend them because of the way minecraft handles block updates (it only updates blocks directly adjacent to the block that has changed) This makes it so that if any block adjacent to the piston changes in any way the piston will update and realise that it's being powered and extend. When it extends it breaks the signal powering the block diagonally above it but, as I said, this won't update it so it won't retract until something near it updates. So what happens is that it starts powered but retracted. When night falls the water turn the grass block into dirt which causes the piston to update and extend. When the morning comes the grass on the sides will eventually spread to the dirt block causing the piston to update again and retract, starting the cycle all over again. Hope you understand it somewhat better now.
