[QUOTE=_Maverick_;39387790]the only mod* i can get working is TMI I've been trying for two days now, I've been looking at every youtube guide, every blog with detailed instructions and i still can't get ANYTHING to work right i'm just going to sit here and cry, don't mind me[/QUOTE] [url]http://forkk.net/MultiMC4/[/url] Click top left-most button to create an instance Right-click the instance icon, 'edit mods' Should be pretty self-explanatory - you just drag and drop zip archives (any marginally popular mod it's just the archive you download) onto the appropriate tab. Double-click the icon to start the instance, it'll download packages if it needs to then install the mods for you. I haven't encountered a single commonly used mod it didn't work with. With MultiMC you can just keep as many copies of the game as you like, all with different mods.
