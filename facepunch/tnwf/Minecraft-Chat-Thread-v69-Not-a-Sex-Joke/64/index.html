[QUOTE=theaceattourney;38398205]I'm not too sure what mipmapping is, but I'll look into it.[/QUOTE] It will prevent all that weird black swirly stuff on the distant blocks from showing up. Mipmapping is pretty much essential if you're using a texture pack that's higher quality than the default Minecraft textures. Anisotropic filtering is also useful (prevents blurry textures at an angle). It should be noted that mipmapping and anisotropic filtering are relatively old technologies and have a completely negligible effect on any modern GPU's performance. Look up Optifine, you won't regret it.
