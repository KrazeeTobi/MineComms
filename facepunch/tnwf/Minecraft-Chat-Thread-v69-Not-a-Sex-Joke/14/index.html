[QUOTE=Zac;37938094]Nope. Still not working. I dont understand, 2:7 is two layers of 7? (Bedrock?)[/QUOTE] It's [url=http://www.minecraftwiki.net/wiki/Flatworld#Preset_code_format]explained in this section of the page[/url], but here's an alternative explanation (although this is the first i've ever looked at this code myself) The "2" is always at the start. The ";" denotes a new section of the customization string. The section after "2;" is where the block definition goes, starting at the bottom of the world and going up. So the first example (2;7,2x3,2;1;village), looks like this: [code]2 ; 1xbedrock,2xdirt,1xgrass ;1;village[/code] As you can see, the 1x/2x/3x just denotes how many layers of that specific block type you want.
