NOTE: Your textures will be wiped if Notch updates terrain.png, so be prepared to re-install packs often. Go to "%appdata%\.minecraft\bin\minecraft.jar" 3. Open "minecraft.jar" with WinRAR 4. Create a folder somewhere called "Minecraft Textures" 5. Extract all png/gif files in there, as well as the folders: "armor" "art" "gui" "item" "misc" "mob" "terrain" to the folder you created. 6. Edit them and save. 7. Drag and drop everything from your "Minecraft Textures" folder back into the jar file that is opened with WinRAR (following the same folder structure) and let them replace the others. 8.
