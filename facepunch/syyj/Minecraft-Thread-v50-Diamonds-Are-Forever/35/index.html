If anyone wants it, here you go, you should probably change the last line to point to wherever your Minecraft executable is though. Oh, and make a new set of bin and saves folders. [release] [B]Instructions:[/B] 1) Place it anywhere. I mean anywhere. It does not matter a ounce, 2a) If you are using a modded .jar already, rename your bin and saves folders to binMod and savesMod, 2b) If you are using vanilla Minecraft rename your bin and saves folder to binVanilla and savesVanilla, 3) Create a new bin and saves folder (or just run the game once), let it update, 4) Run the batch file, answer the choice, 5) Enjoy your choice of game. [B]Code:[/B] [code] :: Stop the system spitting out shit we don't want to see @ECHO off :: Stick us in the Minecraft directory, saves some time later. CD %Appdata%\.minecraft\ CLS :: Gentlemen, BEHOLD! ECHO /--------------------\ ECHO --Minecraft Launcher-- ECHO ------Created by:----- ECHO --------hexpunK------- ECHO \--------------------/ :: Let the user decide (modular too!) CHOICE /T 10 /D N /M "Launch Minecraft ModLoader?" :: Load Vanilla Minecraft IF ERRORLEVEL 2 GOTO Minecraft :: Load ModLoader IF ERRORLEVEL 1 GOTO ModLoader :: This bit is for the ModLoader loading, checks for certain folders to see if it even needs to run. :MODLOADER ECHO Looking for Minecraft Mod Folders... IF EXIST binVanilla GOTO END IF NOT EXIST binMod GOTO MODLOADERERR ECHO Found modded binaries... IF NOT EXIST savesMod GOTO MODLOADERERR ECHO Found modded savedata... GOTO MODLOADERREADY :: Oh dear, we broke something. :MODLOADERERR ECHO Modded Folders not found! Try running default Minecraft. GOTO END :: Renames the folders it needs to launch Minecraft. :MODLOADERREADY ECHO Found mod folders! ECHO Saving old binaries/ saves... REN bin binVanilla REN saves savesVanilla ECHO Preparing modded binaries/ saves... REN binMod bin REN savesMod saves GOTO END :: Launch the good old vanilla Minecraft. Same as before, check for certain folders. :MINECRAFT ECHO Looking for Default Minecraft Folders... IF EXIST binMod GOTO END IF NOT EXIST binVanilla GOTO MINECRAFTERR ECHO Found default binaries... IF NOT EXIST savesVanilla GOTO MINECRAFTERR ECHO Found default savedata... GOTO MINECRAFTREADY :MINECRAFTERR ECHO Default Folders not found! Try running ModLoader. GOTO END :MINECRAFTREADY ECHO Found default folders! ECHO Saving modded binaries/ saves... REN bin binMod REN saves savesMod ECHO Preparing old binaries/ saves... REN binVanilla bin REN savesVanilla saves GOTO END :: Launch Minecraft. Should only happen once everything checks out. :END ECHO Launching Minecraft :: You can change this to whatever you feel. But I assume most of you stick Minecraft on your Desktop. %userprofile%\Desktop\Minecraft.exe ECHO Tip: Do not close this until Minecraft is closed. PAUSE [/code] [/release] I know it isn't much. But someone is bound to have a use for it. When changing your Minecraft executable location, if it is in your user area, keep %userprofile%, it dumps you in your user folder (for me it would be Jordan), then navigate to Documents or whatever.
