[QUOTE=Ninjarooster;26538831]When Notch "implemented" the mods and skins option on the start menu, does it actually work yet? Could someone tell me how or link me to a tutorial?[/QUOTE] It works, but this was written a while ago so I don't know if I've been ninja'd or not. [url=http://painterlypack.net/files/PainterlyPack.zip][img]http://painterlypack.net/resources/title.jpg[/img][/url] 1. Click on the picture to download the Minecraft Painterly Pack, which I'll be using as a demonstration. Make sure the pack you are downloading is compatible with the current Alpha version. Any untextured files will use the original texture. 2. Save the file onto anywhere in your computer. In Firefox the files automatically go into your "Downloads" folder under your username. You can click on "Open file location" as an easy shortcut. In IE I think Windows Explorer opens the downloads folder automatically, but I can't be so sure since I haven't used it in so long. [IMG]http://i.imgur.com/Gca4f.png[/IMG] 3. Open your start menu, search for "run", and open the application. Type "%AppData%" into the search bar. Hit enter and you'll find an otherwise hidden AppData folder. Open "Roaming" if it isn't automatic. [IMG]http://i.imgur.com/PDzZF.png[/IMG] 3. Open .minecraft and open texture packs. Copy and paste the PainterlyPack file into the texturepacks folder. [IMG]http://i.imgur.com/2BE8d.png[/IMG] 4. Open Minecraft, the application. Click on the texture pack. It should be ready to use. [IMG]http://i.imgur.com/lM4JY.png[/IMG] 5. I don't want to rewrite this all, but if you don't feel like going through your files again, there's a shortcut on the screen. Getting a skin is easy, but Notch hasn't implemented it into the actual game yet. You must log in in order to use your new skin. [url]http://www.minecraftskins.com/[/url] is a good site to go to to retrieve skins.
