But can only be inside the wall.[/QUOTE] Here's the bare bone of the one switch to open 2 mirrored doors: [IMG]http://img694.imageshack.us/img694/2710/15580368.png[/IMG] [IMG]http://img808.imageshack.us/img808/5821/11570182.png[/IMG] if you keep this format, you can extend wires and place a floor lower, it should work. If you are interested, this is how it works. For the double doors, one of the doors is mirrored, which caused it to have a reverse value for being open than the door next to it (so when both doors are 'closed', one door is actually closed, the mirrored one is open, but that is to make it have the double door look). if you were to normally wire these, on would cause one door to open and the other to close, so you need a "not" gate, which is made with the 'thermite' torch on the side of the block, which basically inverses the value of the input switch, giving the door the inverse output. The inverse output with the door's reverse code, causes it to behave like the other door.
