I've made some sort of flip flop after spending ages trying to get it negative-edge triggered. [img]http://i45.tinypic.com/2h4w6zp.png[/img] Dark red - wire Red - signal torch (attached to adjacent block) Brown - block (I used dirt) Green - switch Blue - reset White - output In its starting state, putting an input into the green spot (the input needs to be sourced from a signal torch exactly on the green spot, so you will need a double - not before then to keep your input intact) will do nothing, but then switching off that input will flip it so the output (white) will be on. Turning the switch on or off will do nothing, until an input is put into the reset (blue). You should be able to make a binary counter from one of these.
