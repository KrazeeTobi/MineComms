It should be in a .rar file. #2: Open the texture pack in WinRAR. #3: Go to C:\Users\(Your Name)\AppData\Roaming\.minecraft\bin and right click minecraft.jar --> open with --> WinRAR #4: The files in the texture pack should be in a structure with terrain.png in the root folder (it should be visible when you open the .rar) and folders such as armor, mobs etc. #5: If #4 is true, Drag/Drop the texture files from the pack to minecraft.jar (open in winRAR) #6: Run minecraft!
