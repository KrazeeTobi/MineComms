I have to start labeling my caves because they're starting to get so big (I just discovered one that every single time I put down a torch, I discovered a new branch). Each center of the cave system, which is usually where your entrance is, is the hub. You label them Hub A, Hub B, Hub C, etc. You could also add Mine 1 or Mine 2 before that if you have more than one mine. Then, each branch off of the hub is a number, 1, 2, 3, etc. Each branch off of a branch is a Roman numeral. So, the third branch off of the seventh branch off of Hub B in Mine 2 would be: 2B7III I've begun labelling all of my caves like this and so far it has been very rewarding as I no longer get lost.
