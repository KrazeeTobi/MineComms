[QUOTE=Black-Ice;28298143]Can anyone tell me what F3 does? And how to use it to find mobs? People use 'F3' and find mobs but I don't see how it does anything[/QUOTE] F3 is basically mainly used as a way of: - Telling your current position to A LOT of decimal places - Reading your current FPS and how much of your CPU is in use by Minecraft - Reading how well your computer is handling performance-wise (Big red and green lines in the bottom left) But as well as those features, it also adds nametags to specific entities when active, these entities are all the mobs e.g. Cows, Sheep and other friendly mobs as well as Skeletons Creepers e.t.c, each entity is giving a unique ID when spawned, usually a numeric value, and this numeric value is displayed as a nametag above each entity, and is useful for finding mobs, of course you don't know what the mob is, just that there is a mob where the nametag is, however it can be used to put yourself on a state of alert, if you're in a cave, and then you use F3 and see a nametag behind a block of smoothstone, this will obviously be a hostile mob, so it helps in survival, but most actual Minecraft players consider it to be "Cheating" [b]Edit[/b] Damn you guys, I spent my time writing that descriptive paragraph only to find you ninja'd me :argh:
