Hey there Minecrafters, I'm not sure if you know this but recently I found an amazing tip to help performance greatly! (Only applies to multi-core processor systems) -Open Minecraft -Open task manager and right click javaw.exe -Click 'Set Affinity' -Now, make sure only [B]one[/B] core is selected -Press okay This is because Minecraft has horrible multi-core support, having only one core selected makes the game run great. Now enjoy your (near) lag-less Minecraft!
