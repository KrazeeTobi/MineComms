So, this was posted on reddit a couple hours ago and I think it's a GREAT idea. Dear Mojang. If lightning can light trees on fire, try to make it so they don't create large-scale deforestation. Although this WOULD be realistic, it would mean a lighting storm kills off all the trees in a (insert large number)-chunk radius. Also, since when has minecraft been realistic? And then someone replied with The way for it to work is like in real life- if a forest has been devastated by a fire, a bunch of saplings take advantage of the now-rich soil to grow over the same area. How to implement this: When leaves are burned, they drop saplings. When saplings are left alone for a certain time, they plant themselves in the ground. Then when a forest fire ravages an area, yeah, it'll be destroyed for a while, but since each tree is likely to drop more than one sapling (although some of them will probably burn up), you'll end up with more trees. I think that would be interesting to implement, and would add a ton of realism to the game.
