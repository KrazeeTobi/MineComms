If it tries to spawn a puddle where there's already an existing puddle, you just increment the height (It'd then flow to surrounding blocks naturally) Then after the storm is passed, you'd have the normal decay loop go through and make the puddles dry up my decrementing the height value (so the puddles wouldn't instantly vanish, they'd slowly dry up).
